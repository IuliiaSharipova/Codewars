For a given list of digits 0 to 9, return a list with the same digits in the same order, but with all 0s paired. Pairing two 0s generates one 0 at the location of the first one. Examples input: [0, 1, 0, 2] paired: ^-----^ -> [0, 1, 2] kept: ^ Notes: Pairing happens from left to right. For each pairing, the second 0 will always be paired towards the first ( right to left ) 0s generated by pairing can NOT be paired again ( void where not applicable: )
